# Contributing

Thanks for your interest in contributing! This repo hosts a FastAPI + React monorepo.

## Branching
- `main`: stable
- `feat/*`: new features
- `fix/*`: bug fixes
- `docs/*`: documentation changes

## Commit style
- Conventional commits recommended:
  - `feat: add task board drag & drop`
  - `fix(api): normalize due_date formatting`
  - `docs: add getting-started guide`

## Pull requests
- Keep PRs focused and small
- Include screenshots for UI changes
- Describe testing steps and expected behavior

## Running locally
- Follow `docs/getting-started.md`

## Code style
- Backend: `black`, `isort`
- Frontend: TypeScript + ESLint (via CRA)

## Opening issues
- Provide steps to reproduce
- Expected vs actual behavior
- Logs or screenshots when possible
